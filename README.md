# Arduino-Python-Voice-command
Αυτό το έργο παρουσιάζει ένα απλό κυβερνοφυσικό σύστημα όπου ένα πρόγραμμα Python που εκτελείται σε έναν υπολογιστή επικοινωνεί με ένα Arduino UNO μέσω σειριακής επικοινωνίας για τον έλεγχο πραγματικών ηλεκτρονικών εξαρτημάτων.

Το σύστημα χωρίζεται σε δύο κύρια μέρη:

Python (Ελεγκτής PC): Στέλνει εντολές κειμένου στο Arduino και εμφανίζει απαντήσεις.

Arduino (Ελεγκτής Υλικού): Λαμβάνει εντολές, ελέγχει εξαρτήματα υλικού και αναφέρει την κατάσταση του συστήματος πίσω στον υπολογιστή.

Η επικοινωνία υλοποιείται χρησιμοποιώντας σειριακές εντολές USB και απλές εντολές συμβολοσειράς.

Απαιτούμενος Εξοπλισμός:

Arduino UNO (ή συμβατή πλακέτα)
Καλώδιο USB
Υπολογιστής με εγκατεστημένο Python 3
Ηλεκτρονικά εξαρτήματα:
2 × LED
1 × Buzzer (ενεργός)
2 × αντιστάσεις 220Ω (για LED)
Πλακέτα ψωμιού και καλώδια jumper

Αρχιτεκτονική συστήματος
Python (Ελεγκτής υπολογιστή):
Ανοίγει σειριακή σύνδεση με το Arduino
Αποστέλλει εντολές ως συμβολοσειρές κειμένου
Λαμβάνει και εμφανίζει απαντήσεις Arduino
Παρέχει μια απλή διεπαφή βασισμένη σε μενού για τον χρήστη

Arduino (Ελεγκτής υλικού):

Ακούει εισερχόμενες σειριακές εντολές
Χρησιμοποιεί λογική υπό όρους (if / else) για την αναγνώριση εντολών
Ελέγχει LED και βομβητή με βάση τις εντολές
Αποστέλλει μηνύματα επιβεβαίωσης και κατάστασης πίσω στην Python

Υποστηριζόμενες εντολές:
ON	Turns ON all components (LEDs and buzzer)
OFF	Turns OFF all components
STATUS	Returns the current state of each component

Πώς να εκτελέσετε το έργο:
Arduino:
Ανοίξτε το Arduino IDE
Ανεβάστε το παρεχόμενο σκίτσο Arduino στην πλακέτα
Διατηρήστε το καλώδιο USB συνδεδεμένο

Python:
installed required library:
pip install pyserial
run the python script:
python main.py
